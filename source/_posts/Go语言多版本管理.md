官方的库：https://pkg.go.dev/golang.org/dl
在这里可以看到如何使用和安装go语言多版本，不过这个好像对go版本是有要求的，最低是1.22还是多少来着。

另外go install 和 go get 的区别可以看下，因为很多人文章推荐的都是 go get。但是实际上好像不行得go install ，而且go官方推荐的也是go install。两者的区别好像是下载的位置不一样。
